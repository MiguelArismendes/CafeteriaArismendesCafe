# Projeto Arismendes Café

Este é um projeto de website para uma cafeteria chamada Arismendes Café. O site foi desenvolvido utilizando as seguintes tecnologias:

- **HTML**: Estrutura básica do site.
- **CSS**: Estilização e layout.
- **Bootstrap**: Framework CSS para facilitar o desenvolvimento responsivo.

## Funcionalidades

- **Home**: Exibe os itens disponíveis no cardápio, com imagens e descrições.
- **Sobre**: Fornece informações de contato da cafeteria.
- **Produtos**: Permite que os clientes façam reservas online.
- **Loja**: Mostra imagens do ambiente e dos produtos.

## Como Executar o Projeto

1. Clone este repositório para o seu computador.
2. Abra o arquivo `index.html` em um navegador web.
